# 架空のデザイン制作会社「DIGSMILE」のコーポレートサイト

架空のデザイン制作会社「DIGSMILE」のコーポレートサイトです。  
Adobe XDのデザインカンプから静的コーディングしました。


## 特に力を入れた点
ハンバーガーメニューのアクセシビリティ対応に注力しました。  
ボタンには`aria-expanded`、メニューには`aria-hidden`を設定し、JavaScriptでこれらの属性値を動的に更新することで、スクリーンリーダー利用者にもメニューの開閉状態が正しく伝わるようにしています。  
WAI-ARIAに基づいた実装を行うことで、誰にとっても使いやすいUIを意識しました。


## ページ構成
- トップページ（`/`）
- 下層ページ×2
  - Aboutページ（`/about`）
  - Contactページ（`/contact`）


## デザインカンプ
`Adobe XD`


## 使用した技術
- HTML/CSS/JavaScript
- Sass（Dart Sass）
- CSS設計（BEM/FLOCSS）


## 実装した機能およびUIパーツ
- ハンバーガーメニュー
- スティッキーヘッダー
- スムーススクロール
- 入力フォームの必須項目がすべて入力されるまで送信ボタンを非活性化
- 入力フォームの入力情報をGoogle Formsに非同期送信（Ajaxを使用）
